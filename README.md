# SouthParkFrameWork

![South Park Banner](https://etgeekera.com/wp-content/uploads/2013/05/south-park-the-stick-of-truth-banner.jpg)

SouthParkFrameWork is a simple modding and research framework for South Park: The Stick of Truth.

## Features

- DLL framework
- Debug logging
- DirectX9 overlay
- Hooking support
- Runtime patching
- Modding tools

## Requirements

- Windows 10 / 11
- Visual Studio
- DirectX 9 SDK
- Legally owned copy of the game

## Disclaimer

This project is for research, learning, and modding purposes only.

No game files or copyrighted assets are included.

## Status

Work in progress.
